
🎬 Movie Library
Личное приложение для ведения коллекции фильмов.

Автор: [Гергов Марат]

Описание
Movie Library — это простое десктопное приложение с графическим интерфейсом, написанное на Python. Оно позволяет создавать и управлять собственной базой фильмов, хранить информацию о них, а также быстро находить нужные картины с помощью фильтрации.

Все данные (название, жанр, год, рейтинг) сохраняются в локальный файл movies.json, что гарантирует их сохранность после закрытия программы.

Возможности
✅ Добавление фильмов: Удобная форма для ввода данных о новом фильме.
✅ Валидация данных: Приложение проверяет корректность ввода. Год должен быть числом, а рейтинг — находиться в диапазоне от 0 до 10.
✅ Фильтрация: Быстрый поиск по жанру и/или году выпуска.
✅ Автоматическое сохранение: Данные сохраняются в файл movies.json автоматически при каждом добавлении фильма.
✅ Надежность: Обработка ошибок при чтении и записи файла данных.
Технологии
Язык программирования: Python 3.x
Библиотека GUI: Tkinter (входит в стандартную библиотеку Python)
Формат данных: JSON
Инструкция по запуску
Установите Python 3. Убедитесь, что у вас установлена версия Python 3.6 или новее. Скачать её можно с официального сайта: python.org.
Скачайте проект. Склонируйте этот репозиторий или скачайте ZIP-архив с исходным кодом.
Запустите приложение.
